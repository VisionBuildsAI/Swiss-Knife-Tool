
# Shift Master
ShiftMaster is a Python-based tool designed to encrypt and decrypt text using the Caesar Cipher technique. Itâ€™s simple, fast, and perfect for learning basic encryption methods or integrating into larger projects.




## Features

- Encrypt text using a shift-based cipher.
- Decrypt text back to the original message.
- Input validation to prevent errors.
- Easy-to-read and modular Python code.

## Usage/Examples

```python
from shiftmaster import caesar  

Encrypt encrypted_text = caesar
("hello world", 3, encrypt=True)
Decrypt decrypted_text = caesar(encrypted_text, 3, encrypt=False)
```
